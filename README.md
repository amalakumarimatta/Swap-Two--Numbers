# Swap-Two--Numbers
This repository contains multiple ways to swap two numbers using Python programming language.
# Python Number Swapping Techniques

## Description
This repository demonstrates multiple ways to swap two numbers in Python.  
Each method explains a unique approach to improve understanding of Python concepts and programming logic.

---

## Implemented Methods

### • Temporary Variable Method
Swaps values using an extra variable for temporary storage.

### • Tuple Unpacking Method
Uses Python’s built-in multiple assignment feature for direct swapping.

### • Addition and Subtraction Method
Performs swapping using arithmetic operations without an extra variable.

### • Multiplication and Division Method
Uses multiplication and division operators to exchange values.

### • XOR Bitwise Method
Applies bitwise XOR operation to swap integer values efficiently.

### • Function-Based Swapping
Encapsulates swapping logic inside a reusable function.

### • Lambda Function Method
Uses an anonymous lambda function for concise swapping.
